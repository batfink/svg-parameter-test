SVG med with fallback colour. The background should be orange if the browser supports javascript in svg, or else it’s black.

<object width="100" height="100" type="image/svg+xml" data="graphic.svg"></object>

Object-tag with html parameters. The background should be maroon.

<object width="100" height="100" type="image/svg+xml" data="graphic.svg">
    <param name="color" value="maroon"/>
</object>

Object-tag with url-paramers. The background should be teal.

<object width="100" height="100" type="image/svg+xml" data="graphic.svg?color=teal"></object>

Custom element extending svg. The background should be pink. (Doesn’t work with WebReflections polyfill)

<svg is="custom-svg-icon" background-color="pink"></svg>

Custom element extending html with svg as first child. The background should be violet.

<custom-html-icon background-color="violet"></custom-html-icon>
